# Документация к телеграмм-боту для изучения английского языка по карточкам (далее - телебот):

## Название телебота: VES_eng_TBot

*Функционал:*

  Телебот предлагает случайное русское слово из базы данных и 4 варианта его перевода, 
представленные в виде кнопок.

  При нажатии на верный перевод телебот выдаёт сообщение "Отлично!", 
при неправильном переводе - "Допущена ошибка! Попробуй ещё раз вспомнить слово ..."

Дополнительные кнопки:
- "Дальше" - предлагает новое случайное слово для изучения.

- "Добавить слово" - добавляет новое слово, закреплённое за данным пользователем, в базу данных.
Необходимо ввести через пробел русское слово и его перевод.
Если данного русского слова нет в общей базе данных и базе данных данного пользователя, 
то оно добавляется в базу.

- "Удалить слово" - удаляет слово, закреплённое за данным пользователем.
Необходимо ввести русское слово, которое хотите удалить.

### Телебот оперирует базой данных, состоящей из трёх таблиц:
*Таблица user (список пользователей)* имеет поля
  - id (PK),
  - name (имя пользователя),
  - user_id (идентификатор пользователя Телеграмм).

*Таблица db_words (список слов)* имеет поля
  - id(PK),
  - rus_word (русское слово),
  - eng_word (английское слово),
  -  user_id(идентификатор пользователя, FK (таблица user PK, отвечающий user_id)).
    
*Таблица bot_logger* имеет поля 
  - id (PK),
  - date_time (текущие дата и время),
  - user_id (идентификатор пользователя),
  - rus_word (русское слово),
  - operation (test - тестирование, addw - добавление слова, delw - удаление слова),
  - status (True - успешно, False - ошибка).
